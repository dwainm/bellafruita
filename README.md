# Bella Fruita - Apple Sorting Machine Feeder

Industrial conveyor control system for automated apple sorting operations.

## Overview

This Python application controls the conveyor feeder for an apple sorting machine. It communicates with Procon Modbus terminals to manage conveyor operations, ensuring precise material handling for the sorting process.

## Technology Stack

- **Python**: Core application logic
- **PyModbus**: Modbus communication protocol implementation
- **Procon Modbus Terminals**: Industrial control hardware interface

## Features

- Real-time conveyor control via Modbus protocol
- Integration with Procon industrial terminals
- Automated material feeding for sorting operations

## Requirements

- Python 3.x
- PyModbus library
- Procon Modbus terminal configuration

## Installation

```bash
pip install pymodbus
```

## Usage

Configuration details and connection parameters for the Procon Modbus terminals should be set according to your specific hardware setup.
